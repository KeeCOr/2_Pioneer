## 1. 프로젝트 개요

### 1.1 배경
- 웹 브라우저에서 바로 플레이 가능한 항해 무역 경영 시뮬레이션
- React + Vite + TailwindCSS 기반 싱글 페이지 앱
- 실시간 시장 시세 변동 + 항해 시간 현실감 부여 (기본 배로 가장 가까운 항구 약 1시간)

### 1.2 핵심 목표
1. 현실적인 항해 시간 → 보석(유료 재화)으로 단축 가능한 수익화 구조
2. 함선 다양성 + 승무원 능력치 시스템으로 깊이 있는 육성
3. 정보 구매 시스템으로 전략적 의사결정 유도

---

## 2. 기능 범위 (현재 구현 완료)

### 2.1 지도 시스템
- [x] 세계 항해도 (15개 항구, 전 세계 분산 배치)
- [x] Pan (드래그) + Zoom (핀치/휠) 인터랙션
- [x] 줌 아웃 시 바다 배경 유지
- [x] 배 클릭 → 목적지 항구 클릭으로 항해 시작
- [x] 항해 경로 점선 + 화살표 표시

### 2.2 함선 시스템
- [x] 8종 함선 (슬루프·상인선·카라벨·갤리·다우·정크선·갤리온·프리깃)
- [x] 항구별 전용 판매 배 (예: 다우는 인도양 항구, 정크는 동아시아 항구)
- [x] 함선별 속도·화물량·최대 승무원 차이
- [x] 함선 3종 업그레이드 (돛·화물칸·선원숙소, 각 5레벨)
- [x] 이동 속도: 기본 배 최근접 항구 ~1시간, 보석 1개로 즉시 도착

### 2.3 승무원 시스템
- [x] 승무원 3종 능력치 (항해술·상술·체력)
- [x] 항해술 → 선박 속도 영향
- [x] 상술 → 거래 가격 보너스 (구매가 ↓, 판매가 ↑)
- [x] 최대 승무원 수 12명 상한 (배 종류 + 업그레이드 조합)
- [x] 승무원 고용·해고·배치·하선 (항구 전용)

### 2.4 무역 시스템
- [x] 10종 자원 (향신료·도자기·비단·와인·다이아몬드·해산물·면직물·양털·계피·쌀)
- [x] 실시간 시세 (1시간마다 변동)
- [x] 상술 보너스 적용 구매/판매가 표시

### 2.5 정보 구매 시스템
- [x] 4종 정보 상품 (항구 소문·상인 귀띔·상업 분석 보고서·내부 정보)
- [x] 저급 정보: 적중률 45~55%, 변동폭 15~65금, 반복 구매 가능
- [x] 고급 정보: 적중률 78~92%, 변동폭 60~200금, 항구당 1회
- [x] 정보 = "자원 1종 + 항구 1곳 + 방향(상승/하락)" 형태
- [x] 다음 시세 갱신 시 accuracy 확률로 실제 반영, 적중 여부 표시

### 2.6 항구 제한 시스템
- [x] 승무원 관리 → 항구 전용
- [x] 업그레이드 → 항구 전용
- [x] 정보 구매 → 항구 전용
- [x] 배 구매 → 항구 전용 (항구마다 판매 배 다름)

### 2.7 유료 재화
- [x] 보석 (초기 5개 지급)
- [x] 항해 중 보석 1개로 즉시 도착
