# Pioneer Game — 현행 기획서 (v1.2)

**최종 갱신**: 2026-04-27  
**구현 파일**: `src/App.jsx` (단일 파일, ~1,700줄)  
**현행 버전**: v1.10 포터블 exe

---

## 1. 게임 개요

대항해시대를 배경으로 한 무역 전략 게임.  
리스본에서 출발해 유럽·지중해·아라비아·남아시아·동아시아 24개 항구를 탐험하며  
상품을 사고팔아 부를 쌓는 것이 목표.

---

## 2. 세계 & 항구

### 2.1 지도

- 맵 비율: 100×100 논리 좌표
- 줌/패닝: 핀치(터치), 마우스 휠, 마우스 오른쪽 드래그 줌
- 배 추적(팔로우) 버튼: 클릭 시 선택 배 중심으로 5× 줌
- 항구 클릭 → 항구 시세 패널 오픈
- 배(정박) 클릭 → 배 선택 (항구보다 우선)
- 배(항해중) 클릭 → 배 선택

### 2.2 항구 목록 (24개)

|------|------|

| 지중해 ⛪ | 마르세유·제노바·베니스·트리폴리·이스탄불 |
| 아라비아 🕌 | 알렉산드리아·아덴·두바이 |

| 동아시아 🏯 | 말라카·싱가포르·방콕·광저우·상하이·요코하마 |

## 3. 선박

### 3.1 선종 (11종)

| 선종 | 아이콘 | 기본속도 | 적재 | 최대승무원 | 구매가 |
|------|--------|----------|------|-----------|--------|
| 통통배 | 🚤 | 빠름 | 25 | 2 | 1,000 |
| 슬루프 | ⛵ | 빠름 | 55 | 4 | 3,000 |
| 카라벨 | 🛥️ | 빠름 | 80 | 6 | 8,000 |
| 브리간틴 | ⛴️ | 중간 | 100 | 7 | 12,000 |
| 갤리 | 🚣 | 중간 | 90 | 12 | 14,000 |
| 다우 | 🛶 | 중간 | 85 | 6 | 16,000 |
| 상인선 | 🚢 | 중간 | 140 | 9 | 20,000 |
| 플루트 | 🛳️ | 느림 | 200 | 8 | 28,000 |
| 정크선 | 🏮 | 느림 | 220 | 10 | 32,000 |
| 갤리온 | ⚓ | 느림 | 280 | 12 | 45,000 |
| 프리깃 | 🏴‍☠️ | 매우빠름 | 120 | 12 | 65,000 |

### 3.2 선박 능력치 계산

```
속도 = baseSpeed × (1 + (항법평균-50)/200 + 속도업그레이드×0.15) × 연료보정
적재 = baseCapacity + 화물업그레이드×25
최대승무원 = min(14, t.maxCrew + 승무원업그레이드)
상술% = round((거래평균-50)/2 × 선체보정)
```

- **연료보정**: 연료 <30% → 0.5×, <60% → 0.75×, 이상 → 1.0×
- **선체보정**: 선체 <30% → 0.6×, <60% → 0.8×, 이상 → 1.0×

### 3.3 업그레이드 (3종)

| 업그레이드 | 효과 | 비용(지수) |
|-----------|------|-----------|
| 속도 | 항속 +15%/레벨 | 레벨×증가 |
| 화물 | 적재 +25/레벨 | 레벨×증가 |
| 승무원 | 최대 승무원 +1/레벨 (최대 14) | 레벨×증가 |

업그레이드 패널에서 현재 수치 → 다음 레벨 수치 미리보기 표시.

## 4. 승무원

### 4.1 기본 속성

| 속성 | 범위 | 역할 |
|------|------|------|
| 항법 | 30~100 | 항해 속도에 영향 |
| 거래 | 30~100 | 수수료 감소에 영향 |
| 스태미나 | 30~100 | 폭풍 화물 손실 완화 |
| 수리 | 0~60 | 자동 선체 수리 |

### 4.2 특수 승무원 (희귀도)

| 희귀도 | 출현율 | 특징 |
|--------|--------|------|
| 전설 🌟 | 3% | navBonus+55 또는 tradeBonus+55 |
| 희귀 💜 | 9% | navBonus+50 또는 tradeBonus+55 |
| 고급 💙 | 23% | 지역 전문화 보너스 |
| 일반 | 65% | 기본 |

- 지역 전문 능력: 해당 지역 항로에서 navBonus/tradeBonus 추가 적용
- 고용 비용: 일반 500~2,000금, 특수 1,500~6,500금

## 5. 무역 시스템

### 5.1 상품 (10종)

| 상품 | 저렴한 지역 | 비싼 지역 |
|------|-----------|---------|
| 향신료 🌶️ | 남아시아·동아시아 | 유럽·지중해 |
| 도자기 🏺 | 동아시아 | 유럽·지중해 |
| 비단 🧣 | 동아시아 | 유럽·아라비아 |
| 와인 🍷 | 유럽·지중해 | 동아시아·남아시아 |
| 다이아몬드 💎 | 지중해 | 동아시아·남아시아 |
| 해산물 🦐 | 지중해·유럽 | 동아시아·아라비아 |
| 면직물 📦 | 남아시아 | 유럽·동아시아 |
| 양털 🧶 | 유럽 | 동아시아·아라비아 |
| 계피 🌰 | 남아시아 | 유럽·아라비아·지중해 |
| 쌀 🍚 | 동아시아·남아시아 | 유럽·아라비아 |

### 5.2 가격 구조

- 매입가 = 판매가 = 현재 시세 (가격 차별 없음)
- 거래 시 **수수료 10%** 별도 부과
- 수수료 = `max(0, 10 - floor(승무원거래평균/2))%` → 상술 스킬로 최대 0%까지 감소

### 5.3 시세 변동

- 주기: 매 `3,600 / 게임속도` 초
- 변동량: 지역 가중치 기반 ±랜덤
- 히스토리: 항구×상품별 최대 5개 기록

## 6. 항구 시세 패널

항구 클릭 시 오픈. **좌/우 분할 레이아웃**.

| 영역 | 내용 |
|------|------|
| 헤더 | 항구명·지역뱃지·거래소개장·상승/하락 종수·✕ 버튼 |
| 좌측 (144px) | 전체 상품 목록: 아이콘·이름·현재가·등락률 뱃지. 클릭으로 선택 |
| 우측 | 선택 상품 대형 차트(160px) + 현재가·등락 크게 표시 + 최고가·최저가·변동폭 3칸 + 수수료 안내 |

- 패널 열리면 첫 번째 상품 자동 선택
- 오버레이 클릭 또는 ✕ 버튼으로 닫기

## 7. 시장 팝업 (매입)

항구 정박 후 🏪 시장 버튼 클릭.

| 영역 | 내용 |
|------|------|
| 헤더 | 항구명·보유 금화·✕ 버튼 |
| 화물 바 | 현재 적재량 / 최대 적재량 시각화 |
| 상품 목록 | 상품별 구매 버튼 (1개·5개·10개) |
| 하단 | 연료 보충·선체 수리 버튼 |

- 수수료·매입-판매 구분 없음 (가격 = 현재 시세)
- 루트 레벨에 배치 (맵 이벤트와 간섭 없음)

## 8. 정보 시스템 (시세 예측)

### 8.1 정보 등급

| 등급 | 이름 | 기본 비용 | 정확도 | 변동폭 | 반복 |
|------|------|---------|--------|--------|------|
| 소문 | 거리 소문 | 300금 | 30% | ±15~45 | 가능 |
| 힌트 | 상인 귀띔 | 700금 | 40% | ±25~65 | 가능 |
| 분석 | 상업 분석 보고서 | 3,000금 | 58% | ±60~130 | 1회 |
| 내부 | 내부 정보 | 8,000금 | 72% | ±100~200 | 1회 |

- 가격은 세금 레벨 및 반복 구매 횟수에 따라 지수 증가
- 정보 내용: "N턴 뒤 [항구] [상품] 가격 [상승/하락]"

### 8.2 예측 적용 흐름

- 구매 시 `turnsUntil` 결정 (소문 1~3, 힌트 1~2, 분석·내부 1)
- 매 시세 변동마다 `turnsRemaining` 1 감소
- `turnsRemaining = 0` 도달 시 정확도 확률로 적용 여부 결정
- 최대 30개 보관 (오래된 것부터 삭제)

### 8.3 무료 소문

- **게임 시작 시**: 소문 3개 자동 지급
- **매 시세 변동 시**: 소문 3개 자동 추가 지급

## 9. 퀘스트

### 9.1 일반 퀘스트 (3개 고정 제공)

| 유형 | 내용 | 보상 |
|------|------|------|
| 운송 | 특정 상품 N개를 특정 항구에서 판매 | 금화 |
| 탐험 | 새 항구 N곳 방문 | 금화 + 보석 1 |
| 무역 | 누적 판매액 N금 달성 | 금화 |

- 퀘스트는 시세 변동 시마다 갱신 (3개 유지)

### 9.2 일일 목표 (4개, 매일 자정 리셋)

| 유형 | 내용 | 보상 |
|------|------|------|
| 일일 운송 | 특정 상품 N개를 오늘 안에 판매 | 금화 + 보석 1 |
| 일일 항구 순례 | 새 항구 N곳 발견 | 금화 + 보석 1 |
| 일일 매출 목표 | 오늘 누적 판매액 N금 | 금화 |
| 일일 거래 횟수 | 오늘 판매 N회 완료 | 금화 |

- 난이도는 세금 레벨에 따라 자동 스케일
- 우측 상단 위젯 / 모달로 확인 가능

## 10. 세금 시스템

- 주기: 매 600초마다 세금 부과
- 세금액 = `50 × 1.3^(level-1) + 선박수 × max(20, floor(20 × 1.1^(level-1)))`
- 세금 레벨이 높을수록 퀘스트 목표·일일 목표 난이도 상승

## 11. 저장/불러오기

- 자동 저장: 매 시세 변동 후 localStorage 기록
- 수동 저장: 헤더 💾 버튼
- 게임 시작 시: 저장 데이터 존재하면 "계속하기 / 새 게임" 선택 다이얼로그

저장 데이터:
```json
{
  "saveVersion": "1.1",
  "savedAt": "ISO 8601",
  "gs": { "gold, gems, ships, crew, ..." },
  "priceHistory": {}
}
```
