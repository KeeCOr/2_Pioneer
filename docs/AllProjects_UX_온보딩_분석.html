Section 02
Grade A — 벤치마크
A
GuildManager 길드 경영 React + TypeScript v1.1.76
17단계 INTRO_STEPS + 20개 HINT_STEPS. 전 프로젝트 중 온보딩 커버리지 최고.
잘 되어있는 것
  • 17단계 인트로: 세계관 → 목표 → 각 시스템 순서대로 소개
  • 20개 상황별 힌트: 고용·계약·자원·성장 등 각 메뉴 첫 오픈 시 자동 트리거
  • 일일 목표 + 퀘스트 체인으로 지속적 행동 방향 제시
  • 스토리 이벤트(storyAfter)로 퀘스트 완료에 서사 감부여
남은 개선 포인트
  • 힌트 재열람 방법이 없음 — "?" 버튼으로 언제든 다시 볼 수 있어야 함
  • 인트로 스킵 후 처음으로 돌아오는 UX 없음
  • 자원 고갈 · 용병 전사 같은 위기 상황에 시각 경고 부족
  • Day 1 핵심 목표(첫 계약 수주)가 화면에 핀 고정되지 않음

Grade B — 양호하나 개선 여지
B

18 TUTORIAL_STEPS (2 intro + 7+ system). 전투 메카닉까지 커버하는 가장 깊은 튜토리얼.
잘 되어있는 것
  • 2장 인트로: 세계관(검의 탑) + 목표 선명하게 제시
  • 전투 흐름 / 행동 상성 / 세부 기술 / 거리와 행 / 주사위&크리티컬 등 핵심 메카닉 직접 설명
  • TUTORIAL_KEY로 localStorage 기반 1회 표시 관리
  • variant: 'intro' | 'system' 구분으로 서사 → 시스템 자연 전환
미비한 것
  • 튜토리얼 완료 후 인게임 중 힌트 전혀 없음 — 망각 구간 발생
  • 첫 번째 보상 선택 화면에 선택지 설명 없음 (강화·스킬·아이템 차이)
  • 스테미너/컨디션 실시간 경고 없음 (배틀 중 컨디션 저하 알림)
  • 로그라이트 메타 진행(해금 시스템) 설명 없음

  • 멀티플레이어 로비 씬 구조 분리 (MultiplayerLobbyScene)

장르별 공통 개선 패턴
여러 프로젝트에서 재사용 가능한 온보딩 컴포넌트/패턴입니다. 한 번 만들어두면 유사 프로젝트에 빠르게 적용 가능합니다.

⚔️ 전투/배틀 시스템

첫 전투 시작 전 "행동 흐름 다이어그램" 1장. 전투 중 첫 스킬 사용 시 툴팁 자동 표시. 첫 승리/패배 후 핵심 판단 포인트 요약.

적용 대상: TOS, ChessSummon, DragonTactics, GranRumble, SixteenMythology, YugiohLegend, HeroChronicle

🎮 액션/조작

게임 시작 3초 반투명 오버레이: 주요 키 → 기능 매핑. 첫 사용 행동에 "△" 강조 아이콘. 아무 키 입력 시 자동 닫힘.

적용 대상: InkWarrior, TowerOfCrowns, WesternTrainGuardian, IF, GolemRush, RhythmBlade

🏛️ 경영/보드게임

"지금 당신의 차례입니다. 할 수 있는 것: [A] [B] [C]" 행동 가이드 바. 첫 턴에만 추천 행동 강조 테두리.

적용 대상: Pioneer, LOP, WorldBid, KingsCitadel, RTSLite, GuildManager(보강)

📊 시스템 힌트 (첫 노출)

localStorage 기반 seenSystems Set. 새 기능/탭/패널 첫 오픈 시 인라인 말풍선 툴팁. 재열람: 화면 어딘가 "?" 버튼.

적용 대상: 모든 프로젝트 (GuildManager 패턴 재사용)

🎯 첫 세션 목표 카드

게임 시작 후 화면 한 구석에 고정: "지금 할 일: [목표]" 카드. 달성 시 체크마크 + 다음 목표 자동 갱신.

적용 대상: Pioneer, LOP, InisLand, TL, HeroChronicle, SixteenMythology

🏆 즉각 피드백 (토스트)

주요 행동 완료 시 화면 중앙 상단 토스트. +N 플로팅 텍스트. 1.5~3초 후 자동 사라짐. 오류는 오렌지/빨강.

적용 대상: Pioneer(추가), GuildManager(보강), WorldBid, 모든 경영/전략 장르

📈 진행 가시화

다음 해금/승급까지 진행 바. 현재 스테이지/웨이브/라운드 HUD 고정 표시. 누적 수치 변동 애니메이션.

적용 대상: Pioneer, LOP, PokerStrike, GolemRush, WesternTrainGuardian

⚠️ 위기/경고 신호

체력/자원/내구도 30% 미만 → HUD 색상 빨강 전환 + 펄스 애니메이션. 중요 행동 실패 → 즉시 토스트 (로그에 묻히지 않게).

적용 대상: Pioneer, InisLand, WesternTrainGuardian, TL, HeroChronicle

🔦 스포트라이트 오버레이

반투명 배경 + clip-path 구멍으로 대상 요소만 밝게. pulse 링 애니메이션. 대상 클릭 시 자동 해제.

적용 대상: Pioneer(우선), GranRumble, ElementalCommand, YugiohLegend
