## 작업 목록

### 1. 드래그 수정 (두 차례)

**1차 원인**: `onPointerDown`에 `setPointerCapture` 누락  
포인터가 맵 div 밖으로 나가면 `pointermove` 이벤트가 끊겨 드래그 중단.

**2차 원인 (근본 원인)**: 시장/정보 팝업이 맵 div *내부*에 있었음  
`fixed inset-0` 오버레이는 시각적으로만 분리될 뿐, React 가상 DOM 트리에서는 여전히 맵의 자식.  
팝업의 `stopPropagation`이 맵의 `onPointerDown`을 막아 `setPointerCapture`가 호출되지 않음 → 드래그 불가.

**최종 해결**:
- 시장/정보 팝업을 맵 div 밖(루트 레벨 형제)으로 이동
- `onPointerDown`에서 `e.preventDefault()` 제거 (click 이벤트 차단 부작용 있었음)
- `setPointerCapture` 유지

```js
const onPtrDown = useCallback((e) => {
  if (e.pointerType === 'mouse' && e.button !== 0) return;
  try { e.currentTarget.setPointerCapture(e.pointerId); } catch {}
  // ...
}, [setFollowShip]);
```

### 3. 항구 시세 패널 레이아웃 개선

**전**: 2열 그리드, 각 상품마다 작은 44px 스파크라인 카드  
**후**: 좌측 목록 + 우측 대형 차트 분할 레이아웃

- **좌측 (144px)**: 전체 상품 스크롤 목록 — 아이콘, 이름, 현재가, 등락률 뱃지  
- **우측**: 선택 상품의 160px 대형 차트 + 현재가/등락 크게 표시 + 최고가·최저가·변동폭 3칸 + 수수료 안내  
- 패널 열릴 때 첫 번째 상품 자동 선택  
- `selectedPortRes` state 추가

### 4. 매입/판매가 통일 + 수수료 구조 변경

```js
const calcBuyPrice  = (base) => calcPrice(base);   // 수수료 없음
const calcSellPrice = (base) => calcPrice(base);   // 수수료 없음
const getFeeRate    = (tradePct) => Math.max(0, TRADE_FEE_PCT - Math.floor((tradePct||0)/2));
```

매입가 = 판매가 = 현재가. 거래 시 `TRADE_FEE_PCT(10%)` 수수료가 별도 차감.  
상술 스킬로 수수료 감소 가능.

### 6. 시장 팝업 ✕ 버튼 수정

`e.preventDefault()` 제거 전에는 `pointerdown`에서 click 이벤트가 차단되어  
✕ 버튼과 오버레이 클릭이 동작하지 않았음. preventDefault 제거로 해결.

### 7. 업그레이드 미리보기 값 표시

업그레이드 패널에서 현재 수치 → 다음 수치를 직관적으로 표시.

```jsx
const preview = k === 'speed'
  ? { cur: `항속 +${lv*15}%`,       next: `+${(lv+1)*15}%` }
  : k === 'cargo'
  ? { cur: `적재 ${st.capacity}개`, next: `${st.capacity+25}개` }
  : { cur: `승무원 최대 ${st.maxCrew}명`, next: `${Math.min(14,st.maxCrew+1)}명` };
```
