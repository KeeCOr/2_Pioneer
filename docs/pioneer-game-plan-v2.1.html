1

세금 해금 시스템

💡

세금 레벨 상승 = 더 큰 판에 진입했다는 신호. 압박과 보상이 동시에 온다.

레벨별 해금 테이블

L1~2
품목 밀, 면화, 목재 (기초 3종)  |  선원 일반 등급만
L3~4
품목 + 향신료, 비단, 도자기  |  선원 + 숙련 등급 등장
L5~6
품목 + 상아, 염료, 금속  |  선원 + 전문가 등급 등장
L7~8
품목 + 보석류, 희귀 향료  |  선원 + 전설 등급 등장
L9+
품목 + 독점 품목 (세계 단일 항구 전용)  |  선원 전설 풀 확장
🔒

미해금 품목은 거래 패널에 잠금 아이콘으로 표시. 클릭 시 "세금 레벨 N 도달 시 해금" 툴팁 표시. 레벨 상승 시 로그에 "새 품목 해금!" 강조.

2

정보 시스템 개편

정보 구매 = 특정 항구의 특정 품목 가격 방향성만 예측. 명확한 양수 ROI로 전략적 구매를 유도.

예측 정보 형식

📈 리스본 · 향신료 — 2시간 내 +20~30% 예상  (정확도 65%)
📉 런던 · 비단 — 다음 시세 변동 시 -15% 예상  (정확도 45%)

등급별 상세

등급비용정확도예측 내용
소문 무료 (시세 변동마다 3개 자동) 40% 방향만 (상승 / 하락)
귀띔 300금 60% 방향 + 대략적 폭
분석 1,000금 75% 방향 + 구체적 폭 + 시점
내부정보 4,000금 90% 방향 + 폭 + 시점 + 지속 시간
📊

ROI 설계 — 귀띔(300금) 적중 시 향신료 20개 기준 기대 차익 약 +800금. 대량 거래일수록 등급 높은 정보가 압도적으로 유리.

🎁

이벤트 보상으로 정보 지급 — 이벤트 해결 후 자동 지급. 가벼운 이벤트 → 귀띔 1개 / 심각한 이벤트(해적·대공황) → 분석 1개.

3

이벤트 시스템 개편

방치형 게임과 충돌하는 실시간 팝업을 제거. 이벤트는 접속 시 선택지로 일괄 처리. 항로당 최대 1회, 장거리 항로에 집중 발생.

발생 확률

발생 확률 = 항해 시간(시간) × 8% (최대 80%)
항해 시간발생 확률
1시간8%
3시간24%
6시간48%
10시간+80% (상한)

이벤트 처리 흐름

출발 시 이벤트 발생 판정 (1회)
↓ 발생 → 항해 시간 +1시간 자동 추가
↓ 플레이어 접속 시 선택지 팝업 (유효 시간 1시간)
↓ 선택 → 결과 적용 + 정보 보상 자동 지급
↓ 미선택(1시간 경과) → 랜덤 자동 처리 → 접속 시 "자동 처리됨" 로그 확인 가능

로그라이트 레벨 시스템

매매·세금 납부가 경험치로 연결되어 레벨업 시 Slay the Spire 스타일로 선단 전체를 영구 성장시킨다. 시즌 리셋은 개발자가 설정.

5

퀘스트 실패 패널티 변경

즉각 타격(세금 1회분 추가) 대신, 서서히 목이 조이는 영구 세금 증가. 무작정 수락보다 신중한 선택을 유도한다.

패널티 공식
실패 1회 → 현재 일일 세금 × +2% 영구 추가
누적 상한: +20% (10회 실패까지)

예) 세금 1,000금 기준
  실패 1회 → 1,020금
  실패 5회 → 1,100금
  실패 10회 → 1,200금 (상한, 이후 고정)

패널티 완화 방법

방법효과희귀도
특수 퀘스트 보상세금 패널티 -1회분 감소희귀
세금 레벨 상승 시패널티 누적치 초기화자동

9

세금 미납 처리 개편

금화 부족 시 보석보다 먼저 배를 강제 매각. 가장 저렴한 배부터 구매가 50%에 처분. 배 1척 남으면 매각 중단.

세금 납부 시도
↓ 금화 부족
↓ [1단계] 가장 저렴한 배부터 구매가 50%에 강제 매각 (납부액 충족될 때까지 반복)
↓ 배 1척만 남으면 매각 중단
↓ [2단계] 여전히 부족 → 보석 자동 대납 (보석 1개 = 세금 1회분)
↓ [3단계] 보석도 없음 → 미납 누적 (다음 납부 시 2배 징수)

매각 알림 UI

⚠️ 세금 납부 부족 — [슬루프 2호] 강제 매각
   구매가 3,000금 → 매각가 1,500금 수령
   잔여 세금: 0금 (완납)
⚠️

세금 4시간 전부터 헤더 빨강 전환. 1시간 전 "배 매각 위험" 배지 표시. 플레이어가 미리 수동 매각(50%)도 가능.

10

항구 배달 의뢰 시스템

직접 매매가 아닌 화물칸을 빌려주는 방식. 빈 배로 이동할 때 손실 최소화 + 다양한 항구 방문 자연 유도. 보상은 직접 무역의 약 25~35% 수준.

처리 흐름

항구 도착 → [배달 의뢰] 탭에서 목록 확인
↓ 의뢰 선택 (목적지 · 화물 종류 · 필요 칸 수 · 보상 표시)
↓ 출발 → 항해 중 의뢰 화물이 해당 칸 점유 (나머지 칸은 자유 무역 가능)
↓ 목적지 도착 시 자동 정산 + 30 XP 지급

의뢰 예시

[런던 → 리스본]  면화 15칸   보상: 320금   ⏱ 2h
[런던 → 카이로]  목재 20칸   보상: 580금   ⏱ 6h
[런던 → 베이징]  향신료 8칸  보상: 1,200금  ⏱ 10h  ★ 고수익

상세 규칙

항목내용
보상동일 루트 직접 무역 대비 25~35%
화물칸의뢰 화물 점유, 나머지 칸 자유 무역 가능
실패 조건목적지 변경 시 자동 취소 (보상 없음, 패널티 없음)
의뢰 갱신항구 도착마다 0~3개 새 목록 생성
복수 의뢰칸 수 허용 시 같은 목적지 의뢰 동시 수락 가능
이벤트 연동강행 선택 시 의뢰 화물도 손실 포함. 우회/보석 회피 시 의뢰 유지.
